Одновременно с игрой Миша заинтересовался культурой dota-репа. Эта музыка стала для него источником вдохновения и поддержки. Песни, посвящённые игре, помогали ему чувствовать себя частью большого сообщества единомышленников. Благодаря этому новому увлечению Миша наконец нашел своё место в мире, став яркой и самобытной личностью.</p>
